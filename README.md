# XRPL Insight Engine

> Open-source analytics infrastructure for XRPL issued assets

## Overview

XRPL Insight Engine provides comprehensive, real-time analytics for all XRPL-issued assets. By scanning validated ledgers directly, we discover tokens within minutes of creation and track their lifecycle metrics over time.
